## Var explication
Variable was named by a letter and a number like ``b3`` it was ``Blue led number 3``

The sinification is:
- ``b`` = blue color
- ``w`` = white color
- ``r`` = red color

<br>

- ``1`` = Is the first traffic light
- ``2`` = Is the second traffic light
- ``3`` = Is the third traffic light
